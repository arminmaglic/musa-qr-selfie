# Musa QR Code Generator

This application generates QR codes for Musa Ćazim Ćatić verses or any text content.

## Features

- Generate QR codes for Musa Ćazim Ćatić verses
- Custom QR code generation
- Display QR code with verse text

## Usage

1. Open index.html in a web browser
2. Enter the verse or text you want to encode
3. Generate the QR code
4. Download or save the generated QR code

## File Structure

- index.html: Main HTML page
- app.js: JavaScript functionality for QR code generation
- style.css: Styling for the application
- verses.json: JSON file containing Musa Ćazim Ćatić verses
- template-example.jpg: Template image (not tracked in Git)

## Dependencies

- qrcode.js library for QR code generation
